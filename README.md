# Development

## to install dependencies...
```bash
npm install
```

## to refresh tailwind files...
```bash
npx tailwindcss -i ./static/css/input.css -o ./static/css/output.css --watch
```

## run the project...
```bash
go run ./cmd/main.go
```

# To add a new project...
copy the templates from templates/projects/
